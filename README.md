# תרומה למאגר השאלות (Q&A)

תודה על הרצון לתרום למאגר!

## איך מוסיפים שאלות?

### 1. שכפול הפרויקט

יש לבצע את הסדר פעולות הבא:

* קודם כל עושים Fork לריפו הנוכחי (דרך האתר של גיטהאב)
* אחר כך עושים Clone מהריפו הפרטי שלכם (דגש: לא מהריפוי הנוכחי של מיכאל)
* מוסיפים את הריפו הנוכחי (של מיכאל) כ-upstream
* מעתה לפני כל עריכה שתבצעו יש לעשות PULL כדי לוודא שהלוקאלי שלכם מעודכן עם הריפו הראשי של מיכאל
* רק בשלב הזה הוסיפו תוכן


### 2. הוספת תוכן

בכל מודול קיימים:

* קובץ `questions.md` המכיל את כל השאלות.
* תיקיית `answers/` המכילה את התשובות.
* תיקיית `eng/` המכילה שאלות תשובות באנגלית כהעשרה בלבד. פיטוסי הגדיר עברית בלבד - כי הראיון יתנהל בעברית.

בעת הוספת תוכן חדש:

1. הוסיפו את השאלה לקובץ `questions.md`.
2. צרו קובץ תשובה מתאים בתוך תיקיית `answers/`.
3. הקפידו על שמות ברורים ועקביים בין השאלה לתשובה.

דוגמה:

```text
<module: Linux/Python/AWS/Docker/K8s/Terraform>/
├── eng/
├── questions.md
└── answers/
    └── <answer-file>.md
```

### 3. פורמט השאלות

בקובץ `questions.md`:

```markdown
1. מה ההבדל בין Process ל-Thread?
2. מה ההבדל בין SG ל-NACL?
3. מה ההבדל בין IRSA ל-Pod-identity - at AWS EKS
.
.
.
```

### 4. פורמט התשובות

בקובץ המתאים תחת `answers/`:
למען הבהירות השאלה תירשם שוב במלואה - הפעם עם התשובה.

```markdown
1. מה ההבדל בין Process ל-Thread?
---תוכן התשובה---
2. מה ההבדל בין SG ל-NACL?
---תוכן התשובה---
3. מה ההבדל בין IRSA ל-Pod-identity - at AWS EKS
---תוכן התשובה---
```

### 5. Commit & Push - to your personal forked repo

לאחר סיום העריכה:

```bash
git add .
git commit -m "short explain what was made"
git push    # to your personal forked repository
```

### 6. פתיחת Pull Request
* היכנסו לאתר של גיטהאב ומשם בצעו PR (Pull Request) - מול המאגר הראשי (של יובל ומיכאל)
* תארו בקצרה אילו שאלות ותשובות נוספו או עודכנו.

## כללי תרומה

* עברית בלבד כי הראיונות יהיו בעברית. אנגלית העשרה בלבד
* שאלות מקצועיות ורלוונטיות לנושא המודול.
* השתדלו להימנע משאלות כפולות.
* תשובות צריכות להיות מדויקות, ברורות ומעודכנות.
* יש לשמור על מבנה התיקיות הקיים.
* כל שאלה חדשה חייבת לכלול תשובה מתאימה.

תודה על התרומה והעזרה בשיפור המאגר!
